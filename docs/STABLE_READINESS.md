# Stable Readiness

This adapter remains alpha until both release trust and Codex-native parity are
green. Hecate can already replace the package-manager adapter path with these
Go binaries; stable means the standalone adapter is ready for operators to
depend on directly.

## Required Gate Before `v0.1.0`

- `make release-check` passes on the release commit.
- `make real-cli-smoke` passes on a prepared machine with an authenticated
  Codex CLI.
- The Hecate ACP adapter release-binary smoke passes after Hecate is pinned to
  the candidate release.
- Release archives have checksums and GitHub artifact attestations generated by
  the tag workflow.
- The parity matrix below has no blocking `manual gap` rows.

## Stable Decision

Cut `v0.1.0` only after the unresolved stable gates are either implemented and
tested or explicitly documented as non-goals for the stable adapter contract.
